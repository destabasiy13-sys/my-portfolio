# Basliel Gebre — Portfolio

A single-page personal portfolio site built with React and Bootstrap.

## Sections

- **Hero** — name, title, quick links to Projects/Contact
- **About** — short bio
- **Projects** — featured project (Personal News Dashboard), with placeholder
  space for future projects
- **Skills** — tech stack badges
- **Contact** — email and GitHub links

## Tech Stack

React (Vite), Bootstrap (via CDN, no extra npm dependency needed since this
is a static informational site with no data to manage).

## Running Locally

```bash
npm install
npm run dev
```

## Updating

- Add new projects in `src/components/Projects.jsx` (replace one of the
  "coming soon" placeholder cards).
- Screenshots live in `public/images/`.
